# Exercícios de NodeJS
## Git

Iniciar um repositório

```bash
    git init
```

Ver status

```bash
    git status
```

Commit

```bash
    git commit -m "primeiro commit"
```

Vinculando ao Github

```bash
    git remote add origin https://github.com/gabriellypereir4/node-js-exercicios.git
```

Subindo atualização

```bash
    git push origin main
```
